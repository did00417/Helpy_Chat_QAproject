# 🧪 AI 헬피챗 자동화 테스트 프로젝트

본 프로젝트는** AI 헬피챗(AI Helpy Chat) 서비스**에 자동화 테스트를 적용하여
**QA 업무의 효율성과 제품 품질 향상**을 목표로 합니다.

Selenium + Python + pytest 기반으로 구성되어 있으며,
핵심 기능(로그인, 메시지 입력, AI 응답 검증 등)에 대한 자동화 시나리오를 포함합니다.

또한 본 테스트 코드는 Jenkins와 같은 CI 환경에서도 실행 가능하도록 설계되어,
코드 변경 시 자동 테스트 수행, 테스트 결과 리포트 생성 등
지속적인 품질 관리(CI/CD)를 위한 기반을 제공합니다.</br>

- 프로젝트 기간: 251208 ~ 251224

## ⭐ 실행 방법
1. 환경 설정</br>
- 파이썬 버전 확인</br>
`python --version //3.11.9`</br>
`가상환경 활성화 -> vsCode -> Select Interpreter ->  파이썬 버전 설정 후 venv 설정`

2. 의존성 설치(프로젝트에 필요한 패키지 설치)
`pip install -r requirements.txt`

3. 개인 테스트 브랜치 생성
git check -b feature/name
